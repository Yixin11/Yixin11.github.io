日记20201210

Posted on 2020-12-10 In 心情随笔
一篇简短的记录心情随笔。

有一段时间没有更新这个 blog 了，实际上这个页面更多来讲应该是一个树洞吧。

这两个星期做了几件事，第一件是通读了《Effective C++》，充分感受了一下 C++ 这门语言的各种陷阱。我还记得年初的时候在这本书的开头写下“这本书写的很好，但是依我现在的实力还看不懂”，这次能通读一番，应该实力是有足够提升了吧。

第二件事是刷题。这个解耦分了好几步去做，一部分是在牛客上刷高频，在 leetcode 上按分类刷 easy 和 medium，认真学习一下；第二部分是开始看一些算法的入门书，因为是从很基础的讲，所以难度实际上是没有《Effective C++》高的。

第三部分是再去熟悉一下编译原理。但是我没有选龙书和虎书，而是看的是陈硕的《Linux 高性能服务端编程》。这本书并不是仅仅讲 muduo，也包含了一些 C++ 经验谈和并发编程相关的东西。其实我觉得陈硕这个人比较喜欢装逼，看他的书就清楚了。不过还是有点货的，虽然他的语气中充满了对 C++ 程序员乃至 C++ 的鄙视，但是好在还是指明了一条 C++ 学习之路。

其中有那么一天其实我是想动笔写一点关于 C++ 的东西的，然后画好了大纲，去查资料的时候发现对一些内容的理解不够深，遂作罢。这就是两周没更新的原因。

我以前觉得我非常清晰我想要的东西，但是最近反而感觉非常迷茫。这段时间以及前段时间，我去拜读段永平先生的博客，观看雪球嘉年华，想去理解价值投资。这些大 V 的观点非常犀利，虽然评论都说他们都是偷偷带货（比如林园）。比如段永平先生坚定的买入等于持有，一家公司应该建起高高的护城河，永远都不会卖一家好公司除非遇上了更好的公司；比如巴菲特的财富就是滚雪球，奥义是找到够湿的雪和够长的坡；比如烟屁股交易；比如雨枫的价值投机等等等等。写在这里就有点太废话了。

其实我想表达的是，我想去发现自己的价值，但是却发现自己的价值似乎不值一提。我没法证明自己是有价值的。

我疯狂的看书，却发现不知道的东西越来越多。我也疯狂的刷题，但是也没有发现任何意义。我还是拿不到想要的 offer。

或者都有点怀疑，这份坚持是不是对的。

或者说，我是不是真的配得上这份工作呢？虽然大多是复试后就没有后续了，但是差之毫厘失之千里，还有相当长的一段路要走。

事实上机会有很多，抓住一个就够了。但是下次到来的机会，我又能不能紧紧抓住呢，还是说，应该自己主动去争取一个机会呢？
