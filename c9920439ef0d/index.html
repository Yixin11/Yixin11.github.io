Posted on 2020-11-28 Edited on 2020-12-07 In C++
本系列主要是对 C++ 语言的学习。

这部分还没有更新完成！–> 看相对完善的《计算机进阶：后台开发》

这是一个面向 C++ 的专题，知识的主要来源为《Effective C++》、《More Effective C++》、《STL 源码剖析》、《Linux 高性能服务端编程》和《Effective STL》，根据学习的情况和学习时间可能还会追加《C 与指针》和《深度探索 C++ 对象模型》，辅助会从网上搜索一些博客，以及 Google 和华为的编程规范。

很明显，上面所涉及的资料并不含 C++ 的库或者工程，主要是针对 C++ 语言本身的几部分：

  • C 语言
  • C++ 对象编程
  • 泛型编程
  • STL
  • C++ 并发编程
  • 编译原理
