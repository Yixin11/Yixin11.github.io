附录：计算广告业务介绍

简单解释一下实习时的业务方向：市场增长和计算广告。

市场增长：如果把用户量作为一个容器，那么它将会分为两部分：存量和增量。实习时的部门业务便是利用各种手段做增长，最主要的方式是广告，降低获客成本，提高买量效率，提高 ROI。

计算广告：
（才疏学浅，简单介绍）
互联网目前主要有三种变现方式：以数字内容变现（游戏）、以平台中介变现（淘宝、滴滴）、以数字营销变现（广告）。因为互联网飞速发展，用户也更加多元化，为了应对目前这种海量用户精细运营的平台，就出现了计算广告。

计算广告分为很多类，也有很多指标。平台一般分为两类，一类是提供流量的渠道方，比如微博、b站、taptap；另一类是广告主，也就是投放广告方。我们这边是投放方。

所以计算广告后台相当于两个大型进程在频繁的进行网络通讯，又因为要用户加载页面有时间要求，实时通讯要求也比较高。

我做的是其中一个模块（Real Time API 模块），对接平台只有一个（广点通），广点通是腾讯内部的一个广告渠道，因为是个大平台，流量还是挺大的。
